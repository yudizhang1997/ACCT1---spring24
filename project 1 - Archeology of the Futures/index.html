Archeology of the Futures

Design by Yudi Zhang

Project Description:

It's a little device that has eight buttons on a prototype of an octave keyboard to simulate a mixer. Materials used: ESP32 feather, dupont wire, resistors, buttons, card boards, soldering gun and double sided tape.

Coding Documentation:

first step - assembling

I connect all the materials mentioned above to a breadboard and ready to coding.

second step - arduino coding

I'm using an arduino to write the coding to allow the buttons to control the corresponding audio; and to allow the buttons to start playing the audio file in a loop when pressed once, and to stop playing when pressed again.

thrid step - processing coding

I then use the Processing language to write the code that controls the audio file and plays the audio file based on the signal received from the Arduino. When a string starting with "P" is received, the program either starts playing the corresponding audio file in a loop (if it is not currently playing) or stops playing the audio file (if it is playing). In addition, the program draws a rectangle on the graphical interface for each audio file and visualizes the audio playback state by changing the color of the rectangle (gray for playing, white for not playing).
